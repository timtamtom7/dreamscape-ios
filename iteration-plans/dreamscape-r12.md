# Dreamscape R12 — Dream Sharing & Sleep Correlation

## Goals
- Community dream journal sharing (opt-in, anonymous)
- Sleep quality correlation analysis
- Dream journal export and backup

## Features

### Dream Journal Communities
- Share individual dreams anonymously (user-controlled)
- Browse shared dreams by theme/symbol/mood
- "Similar dreams" matching based on your symbols
- Comment on shared dreams (community interpretations)
- Report inappropriate content

### Dream Map Community Layer
- See where others' dreams take place geographically
- Aggregate "dream hotspots" (fictional and real locations)
- Popular symbols visualization across all users
- Community lucidity statistics

### Sleep Correlation
- HealthKit sleep data integration
- Correlate dream content with sleep stages
- Morning vs. afternoon dream quality comparison
- Sleep debt impact on dream vividness
- REM cycle timing analysis

### Data Portability
- Export all dreams as JSON/Markdown/PDF
- iCloud backup with E2EE
- Import from other dream journals
- Dream journal printing (book format)

## Privacy & Safety
- Anonymous sharing only
- Content moderation pipeline
- Personal data never leaves device unless shared
- Block specific symbols from being shared
- Delete all shared data remotely

## Technical
- CloudKit for community data (no user accounts)
- HealthKit for sleep correlation
- On-device content filtering before sharing
- E2EE for personal cloud backup
